# biu 网络安全工具箱 （其他功能正在添加中）

## 功能：
```
Commands:
  clear       clear the screen
  exit        exit the program
  guess       弱口令爆破器,支持:ssh,ftp,mysql,redis,mssql,postgresql,mongodb
  help        display help
  influx      influx 配置疏忽漏洞利用
  scan        主动扫描端口
  shodan      通过shodan被动扫描目标主机
```
## 使用方法:

```
go run main.go 或 直接运行编译好的可执行文件
```
